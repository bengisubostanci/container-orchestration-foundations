What is Docker & Why Use It?
Docker is an ecosystem that allows us to create, deploy, and run applications easily by using containers. It eliminates the hassle of manual setups and dependency conflicts.

Requirements
OS: Docker Desktop is required for Windows (7, 8, 10 Home) and Mac.

Hardware: Requires a 64-bit architecture with CPU Virtualization enabled in the BIOS.

Components: Installing docker-ce installs both the Docker Client and Docker Server.

The Docker Ecosystem
Docker Client & Server: The tools used to issue commands and manage containers.

Image: A blueprint file containing the application, its configuration, and dependencies (similar to a Class in programming).

Container: A live, running instance of an image (similar to an Object in programming).

Docker Hub / Registry: A cloud-based library for storing and sharing images.

Official Images: Tested, optimized, and maintained by Docker or trusted companies.

User Images: Created by the community, formatted as <username>/<imagename>. Free accounts get unlimited public repositories and one private repository.

Note: Docker Hub is the central go-to marketplace for everything Docker, except for the official documentation.

*Türkçe 

Docker Nedir ve Neden Kullanılır?
Docker; uygulamaları konteynerler (containers) aracılığıyla kolayca oluşturmamızı, dağıtmamızı ve çalıştırmamızı sağlayan bir ekosistemdir. Kurulum süreçlerini ve bağımlılık (dependency) sorunlarını tamamen ortadan kaldırır.

Gereksinimler
İşletim Sistemi: Windows (7, 8, 10 Home) ve Mac kullanıcıları Docker Desktop kullanmalıdır.

Donanım: 64-bit bir işlemci ve BIOS üzerinden Sanal yükseltmenin (CPU Virtualization) aktif edilmiş olması gerekir.

Bileşenler: docker-ce kurulumu yapıldığında hem Docker Client (İstemci) hem de Docker Server (Sunucu) yüklenir.

Docker Ekosistemi
Docker Client & Server: Komutları verdiğimiz ve konteynerleri yönettiğimiz mekanizma.

Image (İmaj): Uygulamayı, bağımlılıklarını ve ayarlarını barındıran şablon dosyadır (Yazılımdaki Sınıf/Class mantığına benzer).

Container (Konteyner): İmajın canlı olarak çalışan bir kopyasıdır (Yazılımdaki Nesne/Object mantığına benzer).

Docker Hub / Kayıt Defteri (Registry): İmajların depolandığı ve paylaşıldığı merkezi kütüphanedir.

Resmi (Official) İmajlar: Docker veya güvenilir şirketler tarafından test edilmiş, güvenli imajlardır.

Kullanıcı İmajları: Topluluk tarafından yüklenir ve <kullanıcı_adı>/<imaj_adı> formatındadır. Ücretsiz hesaplarda sınırsız genel (public) ve bir adet gizli (private) depo hakkı sunulur.

Not: Resmi dokümantasyon hariç, Docker ile ilgili ihtiyacınız olan her şey Docker Hub üzerinde yer alır.
