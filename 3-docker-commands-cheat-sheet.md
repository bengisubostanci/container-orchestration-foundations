İşte bu komut satırı notları için hazırladığım en kısa, pratik ve iki dilli (cheat sheet) formatı:

EN 🇬🇧
Run Nginx:
docker run --name nginx -d nginx:1.14

List Running Containers:
docker ps

Stop Container (by Name or ID):
docker stop <container_name>
docker stop <container_id>

TR 🇹🇷
Nginx Çalıştır:
docker run --name nginx -d nginx:1.14

Çalışan Konteynerleri Listele:
docker ps

Konteyneri Durdur (İsim veya ID ile):
docker stop <container_name>
docker stop <container_id>
