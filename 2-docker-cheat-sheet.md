İşte notlarının en can alıcı noktalarını seçerek hazırladığım, hızlıca göz atmaya (cheat sheet) uygun en kısa hali:

EN 🇬🇧
Why Docker?: Easily runs software without setup/dependency issues.

Requirements: 64-bit OS, CPU Virtualization enabled. Docker Desktop for Win (7/8/10 Home) & Mac.

Key Concepts:

Image: Blueprint containing the app and its dependencies (like a Class).

Container: A running instance of an image (like an Object).

Docker Hub: Central registry to store/download images.

Official: Tested and reliable.

User: Format is <username>/<imagename>. (1 free private repo).

TR 🇹🇷
Neden Docker?: Kurulum ve bağımlılık sorunları olmadan yazılımları kolayca çalıştırır.

Gereksinimler: 64-bit işletim sistemi ve CPU Sanallaştırma açık olmalı. Win (7/8/10 Home) ve Mac için Docker Desktop gerekir.

Temel Kavramlar:

Image (İmaj): Uygulama ve bağımlılıklarını içeren şablon dosya (Class mantığı).

Container (Konteyner): İmajın çalışan canlı örneği (Object mantığı).

Docker Hub: İmajların saklandığı ve indirildiği merkezi depo.

Official (Resmi): Test edilmiş, güvenilir imajlar.

User (Kullanıcı): <kullanıcı_adı>/<imaj_adı> formatındadır (1 adet ücretsiz gizli depo hakkı verir).
